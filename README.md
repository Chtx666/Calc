# 🧮 Qt 简易计算器

[![License: GPL-3.0](https://img.shields.io/badge/License-GPLv3-blue.svg)](LICENSE)

这是一个使用 **Qt + C++** 开发的简易计算器，用于练习 Qt Widgets、信号与槽机制和 GUI 编程。  
项目支持加、减、乘、除四则运算，界面简洁直观，适合初学者练手或快速开发桌面应用。

[项目仓库地址](https://github.com/Chtx666/Calc)

---

## 🔹 项目亮点

- 直观简洁的界面设计  
- 实现基本四则运算  
- 清晰的代码逻辑与 UI 分离  
- 可扩展性高，方便增加新功能（如小数运算、括号运算等）  
- 学习和练习 Qt 信号与槽机制的好项目
