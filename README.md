# 🧮 Qt 简易计算器

这是一个用 **Qt + C++** 写的简易图形界面计算器程序，用来练习 Qt 界面编程、信号与槽机制、UI 设计等基础知识。  
项目采用 Qt Widgets 实现，支持基本的四则运算（加、减、乘、除）。:contentReference[oaicite:1]{index=1}
